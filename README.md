# Proyecto de Criptografía Clásica

Este proyecto consiste en el diseño e implementación de una página web accesible públicamente que permite cifrar y descifrar texto mediante el uso de diferentes algoritmos criptográficos. La aplicación fue desarrollada con fines académicos y permite comprender el funcionamiento básico de los métodos de cifrado a través de una interfaz web sencilla e intuitiva.

## Algoritmos implementados
- Cifrado César
- Cifrado Vigenère
- Cifrado AES
- Cifrado Atbash

## Uso de la aplicación
1. Seleccionar el algoritmo de cifrado o descifrado.
2. Ingresar el texto que se desea procesar.
3. Ingresar la clave o parámetro requerido según el algoritmo.
4. Presionar el botón correspondiente para cifrar o descifrar.
5. Visualizar el resultado generado en pantalla.

## Entrega
El proyecto incluye un video de demostración en el cual se muestra el funcionamiento de la aplicación web, el proceso completo de cifrado y descifrado para cada algoritmo y una breve explicación del código fuente.
